# 小米商城前端页面

## 项目简介

这是一个小米商城的前端页面项目，模拟了小米官方商城的主要功能和界面设计。项目采用纯前端技术实现，包含了完整的商城首页布局和交互效果。

## 项目结构

```
├── CSS/              # 样式文件目录
│   ├── SC.css        # 商品橱窗样式
│   ├── banner.css    # 横幅样式
│   ├── banner2.css   # 小横幅样式
│   ├── common.css    # 通用样式
│   ├── footer.css    # 页脚样式
│   ├── header.css    # 页头样式
│   ├── main.css      # 主内容样式
│   ├── rm.css        # 其他样式
│   └── topnav.css    # 顶部导航样式
├── img/              # 图片资源目录
│   ├── banner2/      # 小横幅图片
│   ├── main/         # 主内容图片
│   ├── 健康儿童/      # 健康儿童分类图片
│   ├── 出行穿戴/      # 出行穿戴分类图片
│   ├── 家电/         # 家电分类图片
│   ├── 手机/         # 手机分类图片
│   ├── 智能路由器/    # 智能路由器分类图片
│   ├── 橱窗/         # 商品橱窗图片
│   ├── 生活箱包/      # 生活箱包分类图片
│   ├── 电源配件/      # 电源配件分类图片
│   ├── 电视/         # 电视分类图片
│   ├── 笔记本平板/    # 笔记本平板分类图片
│   └── 耳机音箱/      # 耳机音箱分类图片
├── index.html        # 主页面
└── README.md         # 项目说明文件
```

## 功能模块

### 1. 顶部导航栏
- 包含商城、MIUI、ToT、云服务、金融、有品等链接
- 登录/注册入口
- 消息通知
- 购物车功能

### 2. 页头区域
- 小米logo
- 商品分类菜单（全部商品分类）
- 主导航菜单（小米手机、Redmi红米、电视、笔记本、家电等）
- 搜索框（带搜索建议）

### 3. 横幅轮播
- 大型横幅展示
- 轮播控制按钮

### 4. 小横幅服务区域
- 保障服务
- 企业团购
- F码通道
- 米粉卡
- 以旧换新
- 话费充值

### 5. 商品展示区域
- 手机分类商品
- 智能穿戴商品（耳机、穿戴设备）
- 笔记本平板商品
- 家电商品

### 6. 商品橱窗
- 商品图片展示
- 商品名称和描述
- 商品价格
- 悬停效果

## 技术栈

- **HTML5**：页面结构
- **CSS3**：样式设计
- **原生JavaScript**：交互功能
- **Iconfont**：图标库

## 页面特点

- **响应式设计**：适配不同屏幕尺寸
- **交互效果**：悬停动画、下拉菜单、搜索建议
- **视觉层次**：清晰的布局结构和视觉引导
- **模块化设计**：代码结构清晰，易于维护

## 如何使用

1. 克隆项目到本地
2. 直接打开 `index.html` 文件即可查看效果
3. 无需安装任何依赖，纯前端实现

## 浏览器兼容性

- 支持Chrome、Firefox、Safari、Edge等现代浏览器
- 建议使用Chrome浏览器获得最佳体验

## 开发说明

- 样式文件按功能模块分离，便于维护
- 图片资源按分类存放，结构清晰
- 代码采用语义化HTML，便于SEO和 accessibility
- 使用CSS Flexbox和浮动布局实现页面结构
- 交互功能使用原生JavaScript实现，无需第三方库
